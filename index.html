A Style-Based Generator Architecture for Generative Adversarial Networks

Generator

stylegan的generator相较于普通的gan，主要增加了一个mapping过程，让网络去学习一个disentangle的从latent space到image的映射。

代码中特殊的地方：

在参数初始化时，stylegan加入了equal learning rate的策略

自定义了三类算子的梯度

  • upsample
  • downsample
  • blur

MLP

使用了多层全连接，输入、隐藏、输出的维度都是一样的（512）。

输入是数量为batch size的latent vector，一般维度为[batch size， 512]

Dense

  • 8 * (Equal Dense + bias + act)

Synthesis

Early Layer

CONST

stylegan以一个[?, 512 ,4, 4]，值全为1的tensor作为输入。在源码中，这里不定义batch size，而是根据输入的latent来定义的batch size

TILE

broadcast，使其和latent vector的数量一致。即[batch, 512] vs [batch, 512, 4, 4]

LAYER EPILOGUE

在完成conv或者upscale_conv之后的操作，噪声和style都是在这一步添加

  • add noise
    不直接加入高斯噪声，而是weight * noise，权重[1, 512, 1, 1]，输入的每个channel都对应一个权重（scalar）

  • add bias
    维度为[1, 512, 1, 1]

  • leaky relu

  • instance normalization

  • style modulate

    • latent transform
      把latent vector用一个全连接从[batch, 512]转成[batch, 2, 512, 1, 1]
    • latent dense
    • add bias

  • modulate
    每一个channel使用两个值，对应scaling和shifting操作

Linear Structure

源码中默认使用linear structure，主要是表现grow这个过程，对应初始的constant输入，有两条线路
每一个分辨率[4, 8, 16, 32, 64, 128, 256, 512, 1024]对应一个image_out。
每一个image_out是由

  • 上一分辨率的image_out使用upscale
  • 当前网络的tensor输出转为rgb

两者进行linear interpolation的结果

用到的结构有
BLOCK

基本的结构，除了第一个分辨率，每一个分辨率对应一个block，经历两次加入style和加入噪声

  • CONV UP

    • conv_transpose/conv+upscale2d
    • blur
    • LAYER EPILOGUE

  • CONV

    • conv
    • LAYER EPILOGUE

  • TORGB

    • conv
    • bias

杂记，一

Posted on 2021-08-28 |

blog创了有两三天了，但周末还不想折腾学习上的东西，想好好抒发一下自己的情绪。

和小严走过很多地方，最喜欢的还是西安。

不知道为什么，西安这个城市给我一种特别的感觉。

它很陈旧，十三朝古都，钟鼓楼，旧城墙；

它很繁华，新一线城市，霓虹灯，回民街；

这种不那么矛盾的矛盾，就类似于每个夜晚来到酒馆的顾客，常常转身按亮自己的手机，深吸一口气，然后回头大声喧闹着，摇着桌上的骰子。

自己只走过一个地方，是从小向往到大的城市，香港。

哦对，还有一个上海。

因为，Hongkong and Shanghai Banking Corporation嘛，这个梗你不懂也无所谓，我懂就行。

小时候知道，香港有亚洲排名第一的大学。

不知道，香港有亚洲名列前茅的人口密度。

小时候知道，香港是国际金融中心。

不知道，香港是一座在专制管辖下的自由之城。

如果你不知道我为什么列出这些矛盾，

因为当我去了梦寐以求的大学之后，我得缩在4平方米的出租屋里，4平方给不了你一扇窥探城市夜景的窗。

因为当我以国家有这样的城市自豪时，反占中，游行，暴乱告诉我，小小的地方也有大大的不同。

我更不知道的是，世事更迭，亚洲第一的大学不再是它，有日，自由之城也会不再是它。

那这座城市，留下的只有八九十年代港星们，拿着有个小尾巴的无线话筒，唱着

“

落寞寂寞的一个夜晚

重投平凡再见梦幻

但愿是潇洒告别

休说可归返

”

PhD申请之路

Posted on 2021-08-27 |

我总共在人生的三个阶段直接或间接地想过读PhD这件事：

第一是在小的时候，难免落俗套地说出了我想要当科学家。科学家嘛，如果只是学习课本上的知识又何谈进步呢。

第二是在大学拿到hku的cs offer时，原本看不见的未来，好似有人凿了个小孔，露出稀稀疏疏的光，告诉你，这里走得通，别再朦朦胧胧懵懵懂懂。刚开始读金融双学位的我，一时又迷失了方向。一边计算机着实有趣，奈何院校背景太差，一边金融倒是王牌，无奈半途入行，基础不牢。想要一个一年半制的授课型硕士作为跳板，申到心目中top的计算机院校，不要太天真。想过不如做斜杠青年，可没想过交叉学科要怎么去读博士。

其实也不是天真，是不够勤奋，没有早早地规划，早早地准备。科研成果为0的本科，到了硕士也只是续了一年命，混了更高的文凭。招PhD不是事业单位，哪怕本科背景差，只要你有足够扎实的paper在手，相信prof和judge panel也会“网开一面”，”一网情深“。否则，追随大流，卷院校，卷GPA吧。这么说起，在高中毕业时，就该开始准备了，而我在高中毕业时，还在规划去云南的旅游路线。

第三是我拿到工作offer之后，满怀欣喜地以为自己的假期开始了。可是闲下来才发现，自己这么多年来，从来不是追求早早停留港湾，而是不断地追求上限。可上限跟人一样是活的，像如今，港三cs的PhD就是我的上限，0科研，一般的院校，一般的绩点，能申上我大概也是运气游戏。但一旦迈入了这个门槛，那我的上限又在哪呢？藤校的博后？之后呢？我对自己的估计是不是又和本科一样，以为一个一年半制度的授课型硕士就能足够我去追求更广阔的天空呢？

我承认，在我硕士期间，我无数次摇摆，我想过去一个冷门方向，比如区块链，我满怀信心地告诉prof，区块链我熟悉，我想读博，prof告诉我本科绩点不够，最后成绩出来B+也让我没了底气去继续联系。我想过继续搞图像，所以我硕士project找了cv的老师，但我没想过这大半年的接触，我只在毕业后的博士招生宣讲会上见过老师真实的音容笑貌，也是刚刚答应给我推荐信却无视我博士申请的老师。

如果说我最大的缺陷，就是浅尝辄止，举棋不定，没有坚定的想法和深入挖掘的心。

倘若本科读了双学位之后，继续做金融科技的方向，现在是不是也在外资投行工作了呢，或许导师也能提供一个博士的机会？

倘若硕士决定做区块链，全力以赴，总结文献，申请RA，继续套磁。Prof Yiu是不是早已松口了呢？

倘若如今，面临着计算机中最卷的视觉领域，我也无畏无惧，继续探索，

未来的某天，我想要的，应该都会有吧。
